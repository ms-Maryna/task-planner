# Task Planner Web Application (Laravel MVC)

## Project Overview
This project is a task management web application built using the Laravel framework and MVC architecture.  
The system helps users organise their tasks, manage deadlines, and track progress in a structured environment.

The goal of the project is to demonstrate server-side development skills using Laravel, including database design, authentication, CRUD operations, and secure application architecture.

---

## Problem Statement
Many users manage their tasks using multiple tools such as notebooks, phone reminders, or different applications. This often leads to missed deadlines and poor task organisation.

The Task Planner provides a central platform where users can create, update, and manage their tasks efficiently.

---

## Target Users

### Guest
- Can view public pages
- Encouraged to register for full access

### Registered User
- Can create tasks
- Can edit tasks
- Can delete tasks
- Can mark tasks as completed
- Can view their personal task list

### Admin (Optional Role)
- Can manage users
- Can monitor system activity
